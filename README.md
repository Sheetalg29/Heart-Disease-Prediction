# ❤️ Heart Disease Prediction App

## Overview

This project is a Machine Learning-powered web application that predicts the likelihood of heart disease based on a patient's health parameters. The application is built using Streamlit and a K-Nearest Neighbors (KNN) classification model.

Users can enter medical information such as age, blood pressure, cholesterol level, chest pain type, ECG results, and other health indicators to receive an instant prediction of heart disease risk.

---

## Features

* Interactive web interface using Streamlit
* Real-time heart disease prediction
* Data preprocessing and feature scaling
* One-hot encoding for categorical variables
* Machine learning model deployment
* User-friendly and responsive design
